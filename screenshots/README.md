# Screenshots Directory

This directory contains screenshots of the Stock Index Pattern Discovery and Prediction application.

## Screenshot Files

Place your screenshots in this directory with the following names:

- `main-interface.png` - Main application window
- `dataset-upload.png` - Dataset upload screen
- `preprocessing.png` - Dataset preprocessing output
- `svm-results.png` - SVM model results
- `random-forest-results.png` - Random Forest model results
- `bayesian-ridge-results.png` - Bayesian Ridge model results
- `xgboost-results.png` - XGBoost model results
- `tpa-lstm-results.png` - TPA-LSTM model results
- `multivariate-lstm-fcn-results.png` - Multivariate LSTM-FCN results
- `rmse-comparison.png` - RMSE comparison graph
- `forecast-7-days.png` - 7-day forecast visualization

## How to Add Screenshots

1. Run the application using `python Main.py`
2. Take screenshots of each feature using Windows Snipping Tool or Print Screen
3. Save the screenshots in this directory with the names listed above
4. The README.md in the parent directory will automatically display these screenshots
